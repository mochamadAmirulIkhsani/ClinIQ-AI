# GLOSSARY.md

* **9Router**: The AI API routing service that manages requests to various LLM backends while providing an OpenAI-compatible interface.
* **Clinical Vignette**: A concise clinical case report that includes demographic data, medical history, presenting symptoms, and diagnostic findings, used as the base for quizzes.
* **ICD**: International Classification of Diseases, the global standard for disease data published by the WHO.
* **Locale Context**: A parameter used to customize AI explanations for specific cultural or behavioral contexts (e.g., `INDONESIA_JAJAN`).
* **N+1 Query**: A performance antipattern where the application makes N additional database queries to fetch related data instead of retrieving it in a single join.
* **SOT (Source of Truth)**: The database or service that holds the master, definitive record of a specific piece of data.
* **Zero-Token Architecture**: An architectural pattern that prioritizes caching results to reduce or eliminate external API token consumption after the initial generation.
